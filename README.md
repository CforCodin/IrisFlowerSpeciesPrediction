# IrisFlowerSpeciesPrediction

<b>Problem statement:</b>
Given sepal length, width and petal length, width of Iris flower and the model have to predict the species based on the variables.
3 flower species are-

1. Iris-verginica
2. Iris-setosa
3. Iris-versicolor
